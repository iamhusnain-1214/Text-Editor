# Vim-Style Text Editor

A fully functional modal text editor inspired by Vim, built in C++.

## Features
- Three modes — Normal, Insert, and Command-Line
- Custom MyString class built from scratch using char* — std::string strictly prohibited
- Full memory management with no STL strings anywhere
- Search and replace, undo/redo, copy/paste
- Word wrap at 70 characters
- File I/O — open, save, save as
